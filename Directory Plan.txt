Planning ahead on how the directory is going to be formed over.
itms = files/folders

Main.py
Basic Functions
  Navigation
    wai.py (outputs working dir)
    lf.py (outputs items in working dir)
    gt.py (changes dir)
    fd.py (finds a itm)
  Manipulation
    rn.py (renames itms)
    dc.py (copies itms)
    mi.py (moves itms)
    ri.py (deletes itms)
    ct.py (creates itm; may be a file or dir)
  Help
    hp.py (shows help or manual of a cmd)
    fc.py (shows cmds with the input funct)
  Files
    Manipulating Contents
      ed.py (text editor)
      kb.py (combines file conts)
      fr.py (finds text in conts)
    dp.py (display conts of a file)
    op.py (opens a itm or prgm)
  Proccess
    qt.py (exits a prgm)
    ei.py (exits sh)
  Other
    mt.py (solves a math expr)
    dl.py (pause for a some time)
    sc.py (creates shortcuts for a cmd)
    ou.py (outputs text)
