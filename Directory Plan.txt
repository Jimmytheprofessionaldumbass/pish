Main.py
Basic Functions
  Navigation
    wai.py (outputs working directory)
    lf.py (outputs items in working directory)
    gt.py (changes working directory)
    fd.py (finds a item)
  Creating Changes
    rn.py (renames items)
    dc.py (copies items)
    mi.py (moves items)
    ri.py (deletes items)
    ct.py (creates item; may be a file or directory)
  Help
    hp.py (shows help or manual of a command)
    fc.py (shows commands with the desired function)
  Files
    Manipulating Contents
      ed.py (text editor)
      kb.py (combines file contents)
      fr.py (finds text in contents)
    dp.py (display contents of a file)
    op.py (opens a item or program)
  Proccess
    qt.py (exits a program)
    ei.py (exits shell)
  Other
    mt.py (solves a math expression)
    dl.py (pause for a certain time)
    sc.py (creates shortcuts for a commands)
    ou.py (outputs text)
